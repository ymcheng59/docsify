把你的整身立绘图片放在这个目录,游戏会自动显示。

【最简单的用法 —— 直接传图就行】
- 默认形象命名为:  base.png
- 其它造型命名为:  look-1.png、look-2.png、look-3.png …(最多到 look-20)
- 支持 .png / .jpg / .webp
- 建议:同一个角色、同样的画布大小/站位,只换服装,这样切换时不会跳动。
- 透明背景或白底都可以。

传完后:
- 如果是传到 main 分支(GitHub 网页上直接拖拽上传即可),网站会自动重新部署,刷新就能看到。

【进阶用法 —— 自定义名字(可选)】
在本目录放一个 looks.json,内容示例:

[
  { "file": "base.png",   "name": "基础" },
  { "file": "look-1.png", "name": "碎花连衣裙" },
  { "file": "look-2.png", "name": "牛仔外套" }
]

有 looks.json 时优先用它(名字、顺序都按你写的来);没有就用上面的自动识别规则。
