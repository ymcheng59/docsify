# ✨ 梦幻换装屋

一个纯前端的换装小游戏，使用 HTML + CSS + 内联 SVG 制作，**无需任何依赖或构建工具**，直接用浏览器打开 `index.html` 即可游玩。

## 玩法

- 在右侧面板切换 **发型 / 上装 / 下装 / 鞋子 / 配饰 / 背景** 等分类
- 点击款式更换造型，点击色块更换颜色
- 🎲 **随机造型**：一键生成惊喜搭配
- ↺ **重置**：恢复初始造型
- 📷 **保存图片**：把当前造型导出为 PNG 图片下载到本地

## 内容一览

| 分类 | 可选项 |
| --- | --- |
| 发型 | 短发、长发、双马尾、丸子头、波浪卷（8 种发色） |
| 上装 | T恤、卫衣、连衣裙、吊带、外套（10 种颜色） |
| 下装 | 半身裙、牛仔裤、短裤、长裙（10 种颜色） |
| 鞋子 | 运动鞋、靴子、高跟鞋、平底鞋（10 种颜色） |
| 配饰 | 蝴蝶结、眼镜、皇冠、帽子、耳环 |
| 背景 | 粉色、天空、薄荷、黄昏、紫梦、星空 |

## 运行

直接双击 `index.html`，或在该目录下启动一个静态服务器：

```bash
python3 -m http.server 8000
# 然后浏览器打开 http://localhost:8000/dress-up-game/
```
