# ✨ 梦幻换装屋

一个纯前端的换装小游戏，使用 HTML + CSS + 内联 SVG 制作，**无需任何依赖或构建工具**，直接用浏览器打开 `index.html` 即可游玩。

## 玩法

- 在右侧面板切换 **发型 / 上装 / 下装 / 鞋子 / 配饰 / 背景** 等分类
- 点击款式更换造型，点击色块更换颜色
- 🎲 **随机造型**：一键生成惊喜搭配
- ↺ **重置**：恢复初始造型
- 📷 **保存图片**：把当前造型导出为 PNG 图片下载到本地

- 💾 **收藏槽**：3 个本地收藏位，点击保存/读取造型，右键清空（数据存在浏览器 localStorage）

| 分类 | 可选项 |

| 下装 | 半身裙、牛仔裤、短裤、长裙、百褶裙、背带裤（可不穿，10 种颜色） |
| 鞋子 | 运动鞋、靴子、高跟鞋、平底鞋、玛丽珍（可光脚，10 种颜色） |
| 头饰 | 蝴蝶结、眼镜、皇冠、帽子、耳环、猫耳、面罩、花环 |
| 装饰 | 项链、翅膀、围巾、小挎包、披风（可与头饰叠加） |

## 本地运行

直接双击 `index.html`，或在该目录下启动一个静态服务器：

```bash
python3 -m http.server 8000
# 然后浏览器打开 http://localhost:8000/dress-up-game/
```

## 在线部署（Vercel）

仓库根目录已提供 `vercel.json`，访问站点根路径 `/` 会自动跳转到换装屋。
在 Vercel 导入本仓库后即可一键部署（纯静态，无需构建命令）。
